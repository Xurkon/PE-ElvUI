# ElvUI (Project Ebonhold Edition)

A complete User Interface replacement for World of Warcraft 3.3.5a, customized for Project Ebonhold.

## Features
-   **Complete UI Overhaul**: Replaces Blizzard's default UI with a clean, modern, and highly customizable interface.
-   **Project Ebonhold Integration**: Specifically tweaked to work seamlessly with Project Ebonhold custom content and frames (e.g., `ProjectEbonholdPlayerRunFrame`).
-   **Performance Optimized**: efficient code ensuring minimal impact on frame rate.
-   **Modular Design**: Enable or disable specific modules (Bags, Chat, UnitFrames, etc.) as needed.

## Installation
1.  Download the latest release.
2.  Extract the `ElvUI` and `ElvUI_OptionsUI` folders to your `Interface\AddOns` directory.
3.  Launch the game and follow the on-screen installation guide.
