Latest Changes

v6.10 - 2026-01-30

  • Critical Freeze Fix: Resolved a complete client freeze that occurred during UI reloads.
  • Refactored Initialization: Removed legacy polling loops (recursive `E:Delay`) that could run indefinitely if the target frame was missing. Module now correctly initializes via `PLAYER_ENTERING_WORLD`.
  • Recursion Guard: Implemented a safety mechanism in `hooksecurefunc` for `SetScale`. This prevents "addon fighting" (synchronous infinite loops) if another addon tries to reset the frame scale in the same execution cycle.
  • Module Fix (SwingBar): Fixed an infinite recursion loop in `ElvUI_SwingBar`. The module's hook on `Update_PlayerFrame` was triggering itself repeatedly; added a recursion guard to prevent client freezes.
